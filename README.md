# 🩺 Diabetes Risk Prediction

Сравнительный анализ интерпретируемых ML-моделей и прототип веб-приложения.

## Запуск
1. `pip install -r requirements.txt`
2. `uvicorn src.api.main:app --reload`
3. Открой в браузере: 
    Приложение: `http://127.0.0.1:8000/app`
    Документация API: `http://127.0.0.1:8000/docs`
